## Do NOT Translate

The following must remain unchanged:

- API paths (`/api/login`)
- URLs
- JSON keys
- Database fields
- Code identifiers
- Environment variables
- Product / Company names

---

## Style Rules

1. Use **繁體中文 (Taiwan)** only.
2. Avoid Mainland Chinese wording.
3. Prefer short UI text.
4. Keep consistency across screens.
5. Do not mix “登入 / 登錄” → always use **登入**.
6. Do not mix “儲存 / 保存” → always use **儲存**.
7. Use “刪除” instead of “移除” unless context requires.
